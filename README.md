# Tic Tac Toe

A text-based Java implementation of the classic Tic Tac Toe game for two players.

## Description

This program allows two users to compete at the game Tic Tac Toe. Players alternate turns, selecting squares numbered 1-9 on a 3x3 grid.

## Features

- Traditional Tic Tac Toe rules enforcement
- Two-player gameplay with alternating turns
- Clear board display showing available moves
- Win detection for rows, columns, and diagonals
- Draw (CAT) detection when all squares are filled
- Input validation and error handling

## How to Play

1. Compile the Java program: `javac TicTacToe.java`
2. Run the game: `java TicTacToe`
3. Players take turns entering square numbers (1-9) to place their marks
4. The game ends when a player wins or all squares are filled (CAT)

## Board Layout

```
1 | 2 | 3
---------
4 | 5 | 6
---------
7 | 8 | 9
```
